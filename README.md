# Ant Design Pro

This project is initialized with [Ant Design Pro](https://pro.ant.design) and customized as the **Ant Design Pro**. Follow is the quick guide for how to use.

## Environment Prepare

Install `node_modules`:

```bash
npm install
```

or

```bash
yarn
```

## Provided Scripts

Ant Design Pro (and this scaffold) provides some useful scripts to help you quick start and build the web project, check code style, and run tests.

Scripts provided in `package.json`. It's safe to modify or add additional script:

### Start project

```bash
npm start
```

### Build project

```bash
npm run build
```

### Check code style

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```

### Test code

```bash
npm test
```
